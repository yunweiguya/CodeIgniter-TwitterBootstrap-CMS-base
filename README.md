Codeigniter Twitter Bootstrap CMS Base Build
============================================

This is a base build Codeigniter project using Ion-Auth auth plugin along with Twitter's Bootstrap CSS.

Install
-------
1. Download project. 
2. Import sql to your mysql db. 
3. Set up VHOST with SetEnv APPLICATION_ENV develop 
4. Adjust DB details in deploy/application/config/develop/database.php
5. Adjust site title in deploy/application/config/develop/constants.php

Default Login:
--------------
Username: admin@admin.com
Password: password
(as per ion-auth)
